# WWR Dashboard
A dashboard GUI for Raspberry pi3 based dashboard for the car FY 2025-26, the dashboard will support and show all the data of the ECU in a way desirable to the driver
The Dashboard being manually coded means, there is room for adjustabiltiy and free updates to the source code. while being easily tested on any machine. 


## Getting started

Clone the repository and keep all the files in same directory
